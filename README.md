# ささっと手軽にアプリを作成！PythonとStreamlitでデータ分析ダッシュボード開発ハンズオン

## ハンズオン手順

1. ライブラリのインストール
```
pip install streamlit
pip install janome
```

2. コードを作成
streamlit_app.pyを使用

3. Streamlitの実行
```
streamlit run test.py
```
