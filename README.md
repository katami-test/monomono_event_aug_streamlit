# ささっと手軽にアプリを作成！PythonとStreamlitでデータ分析ダッシュボード開発ハンズオン

## ハンズオン準備
`monomono_streamlit_testdata.csv`をローカルにダウンロード

## ハンズオン手順

1. ライブラリのインストール
```
pip install streamlit janome
```

2. コードを開く
`streamlit_app.py`を使用

3. Streamlitの実行
```
streamlit run streamlit_app.py
```
4. シンプルブラウザを開く
